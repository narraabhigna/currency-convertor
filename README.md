# Currency Converter

Welcome to the Currency Converter project! This project provides a web application that allows users to convert amounts between different currencies using real-time exchange rates.

## Project Overview

The Currency Converter application enables users to input an amount in one currency and convert it to another currency using up-to-date exchange rates. This tool is useful for travelers, businesses, and anyone who needs to perform quick and accurate currency conversions.

## Features

- **Real-Time Exchange Rates**: Fetches the latest exchange rates from a reliable API.
- **Multiple Currencies**: Supports conversion between a wide range of currencies.
- **User-Friendly Interface**: Simple and intuitive interface for easy currency conversion.

## Technologies Used

- Javascript
- HTML/CSS for the web interface

## How It Works

1. **Input Amount and Currency**: Users input the amount they want to convert and select the source and target currencies.
2. **Fetch Exchange Rate**: The application retrieves the latest exchange rate for the selected currencies from an external API.
3. **Calculate Conversion**: The application calculates the converted amount based on the fetched exchange rate.
4. **Display Result**: The converted amount is displayed to the user.
